# Gephi generative palette names

Gephi generative palette names

## Usage

``` r
gephi_palettes()
```

## Value

Character vector of Gephi generative palette identifiers.

## Examples

``` r
gephi_palettes()
#>  [1] "default"      "fancy_light"  "fancy_dark"   "shades"       "tarnish"     
#>  [6] "pastel"       "pimp"         "intense"      "fluo"         "red_roses"   
#> [11] "ochre_sand"   "yellow_lime"  "green_mint"   "ice_cube"     "blue_ocean"  
#> [16] "indigo_night" "purple_wine" 
```
